WF025 – Simulación de error por restricciones
Acción no permitida
⚠

No se puede completar esta acción debido a una de las siguientes razones:

  • El usuario está vinculado a proyectos, tareas o bitácoras activas.
  • No tiene permisos suficientes para modificar este usuario.
  • El usuario ya ha sido eliminado o bloqueado permanentemente.

Si necesita realizar esta acción, contacte con el Administrador del sistema o revise las dependencias del usuario.
